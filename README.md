# README

This project implements various algorithms and data structures.

Clone and navigate to the repository.

```
git clone git@github.com:fokionzervoudakis/algo-ds.git
cd algo-ds/
```

Build with *bash*.

```
chmod +x build.sh
./build.sh
```
