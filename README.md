# README

This project implements various algorithms and data structures.

## Dependencies

- *Java 12*

## Deployment

Clone the repository and build:

```
git clone git@gitlab.com:fzervoudakis/algo-ds.git
cd algo-ds/
chmod +x build.sh
./build.sh
```
